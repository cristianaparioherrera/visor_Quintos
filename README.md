# Visor Quintos de Mora

![Cabecera Montes y Medio Natural](img/cabecera_montesymedionatural.png.png)

## Visor estático de Quintos de Mora

Este visor permite consultar los resultados obtenidos en el marco del Trabajo Fin de Grado sobre la comparación metodológica del empleo de imágenes espectrales a diferentes escalas para la detección de estrés por herbivoría en matorrales mediterráneos.

El visor permite visualizar de forma interactiva la distribución de los ejemplares vegetales muestreados, los cercados experimentales, los parches hiperespectrales y los valores de distintos índices espectrales asociados a cada planta.
